# CV en ligne — Mohammed Atir

Ce dossier contient un site **HTML/CSS/JavaScript** (sans framework) basé sur les informations de ton CV.

## Ouvrir le site

### Option A (simple)
- Ouvre `index.html` dans ton navigateur.

### Option B (recommandé)
Un petit serveur local évite certains soucis de navigation/ancres selon le navigateur.

- **Avec Python**:

```bash
cd D:\cv-site
python -m http.server 5173
```

Puis ouvre `http://localhost:5173`.

## Fichiers
- `index.html`: contenu (sections CV)
- `styles.css`: design + animations CSS
- `script.js`: animations au scroll, menu mobile, barres de compétences, formulaire e‑mail
